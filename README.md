# C-learning
A collection of C programs and small-scale projects built while mastering
the core concepts of the C language. This repository serves as a personal 
laboratory for testing logic, memory management, and data structures.

Concepts Implemented:
Basics: Data types, operators, and control flow (if-else, switch-case).
Loops & Logic: Implementing iterative solutions using for, while, and do-while.
Functions: Modular code design and scope management.
Pointers & Memory: Manual memory allocation and address manipulation (the "C" specialty).
